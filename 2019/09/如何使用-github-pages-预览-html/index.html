如何使用 GitHub Pages 预览 HTML

September 22, 2019
  1. 进入repo，打开setting。
  2. 往下拉到GitHub Pages。
  3. source一栏选择master branch。
  4. 得到这个repo的预览地址。
  5. 在预览地址后面加上想访问的html路径名称即可。
  6. 例如我的html路径名称为git-demo-1/index.html，则可以访问http://lynnlu.xyz/git-demo-1/index.html。
